# رسیور اکانت تلگرام

یک سیستم حرفه‌ای و مدولار برای دریافت و ذخیره اکانت‌های تلگرام با استفاده از کتابخانه Telethon.

## ویژگی‌ها

- ✅ ورود به اکانت تلگرام از طریق ربات
- ✅ پشتیبانی از رمز دو مرحله‌ای
- ✅ ذخیره سشن‌ها به صورت امن
- ✅ معماری مدولار و قابل توسعه
- ✅ مدیریت خطاها و استثناها
- ✅ لاگ‌گذاری کامل
- ✅ سناریو پیشرفته با حل خودکار کپچا
- ✅ یادداشت‌گذاری برای رباتها

## نحوه استفاده

1. ربات را در تلگرام استارت کنید: `/start`
2. شماره تلفن خود را ارسال کنید (مثال: `+989123456789`)
3. کد تایید دریافتی را ارسال کنید
4. در صورت داشتن رمز دو مرحله‌ای، آن را ارسال کنید (یا `/skip`)
5. سشن شما در پوشه `sessions` ذخیره می‌شود

## استفاده به صورت ماژول

```python
from src.services import AccountReceiver
from src.models import AccountCredentials

# ایجاد نمونه
receiver = AccountReceiver()

# ورود به اکانت
credentials = AccountCredentials(
    phone="+989123456789",
    code="12345",
    password="my_password"  # اختیاری
)

result = await receiver.login_account(credentials)

if result.success:
    print(f"ورود موفق! سشن: {result.session_path}")
else:
    print(f"خطا: {result.message}")
```

برای راهنمای کامل: docs/USAGE.md
